# Mini CRM Dashboard

A full stack Lead Management System built using React, Node.js, Express, and PostgreSQL.

## Features
- Add Leads
- Update Lead Status
- Delete Leads
- Search Leads
- Dashboard Summary

## Tech Stack
Frontend:
- React
- Vite
- Axios

Backend:
- Node.js
- Express.js

Database:
- PostgreSQL (Neon)

## Installation

### Backend
cd backend
npm install
npm run dev

### Frontend
cd frontend
npm install
npm run dev

## API Endpoints

GET /api/leads

POST /api/leads

PUT /api/leads/:id

DELETE /api/leads/:id
